Detailed Description

Represents a pair of distances for walking and driving.

The Distance class encapsulates two distance values: one for walking and one for driving. It provides methods to retrieve each of these values.

Parameters
walkingThe distance value representing the walking distance.
drivingThe distance value representing the driving distance.
